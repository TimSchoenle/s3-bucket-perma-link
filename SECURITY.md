# Security Policy

## Supported Versions

Only the latest released version is supported with security updates.

| Version | Supported |
| ------- | --------- |
| Latest release | :white_check_mark: |
| Older releases | :x: |

## Reporting a Vulnerability

Please report vulnerabilities through GitHub Security Advisories for this repository.
If that is not possible, open a private channel with the maintainers and include
reproduction steps, impact, and affected versions.
